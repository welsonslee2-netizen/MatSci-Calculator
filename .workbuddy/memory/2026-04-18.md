# 2026-04-18 工作记录

## 经典/科学计算器 MathLive 输入修复
- **核心问题**：经典计算器和科学计算器的 math-field 输入框使用了错误的 API（`getValue()/setValue()` 拼接字符串），导致 LaTeX 数学结构被破坏，输入的不是科学表达式而是纯文本
- **修复内容**：
  1. 经典计算器小键盘：所有按钮操作改为 `executeCommand('insert', latex)` — 保持分数/根号/幂次等结构渲染
  2. 科学计算器小键盘：同上，`insert` 命令替代字符串拼接
  3. `latexToMathJS()` 函数大幅增强：支持隐式乘法（3x→3*x）、`\binom`、`\sqrt[n]{}`、`\log_{base}`、双曲函数等
  4. 统一 `clear`/`backspace`/`reset`/`history` 等操作对 math-field 的兼容性
  5. 历史记录同时保存原始 LaTeX 和 math.js 表达式，回填时使用 LaTeX 保持格式
  6. `toggleClassicSign`/`applyPercentToClassic`/`replaceClassicRange` 等函数增加 math-field 分支
- **关键 API**：`field.value`（LaTeX getter/setter）、`field.executeCommand('insert', latex, options)`、`field.executeCommand('deleteBackward')`、`field.executeCommand('deleteAll')`

## 科学计算器 UI 优化（第二轮）
- **双反斜杠问题**：`data-insert` 属性中 `\\frac` 在 HTML 属性中不会被反转义为 `\frac`，导致 MathLive 插入双反斜杠。修复：所有 `data-insert="\\..."` 改为 `data-insert="\..."`
- **布局遮挡问题**：科学计算器 feature-panel 内小键盘太长，滚动时输入框不可见。修复：让 feature-panel 设置 `max-height: calc(100vh - 12rem)` + `overflow-y: auto`，使输入区域和标题行 sticky 固定在面板顶部
- **占位符优化**：`#?` 占位符替换为 MathLive 的 `#0` placeholder 语法，插入时自动创建可编辑占位符
- **修改文件**：`index.html`（双反斜杠修复）、`styles.css`（粘性布局）、`app.js`（占位符处理）

## 计算器表达式计算修复（第四轮）
- **经典计算器括号问题**：函数按钮（sin/cos/tan/ln/log/sqrt）原来用纯文本 `sin(`，MathLive 收到后不是有效 LaTeX。修复：改为 `\sin\left(#0\right)` 等 MathLive LaTeX 命令 + `#0` 占位符，自动创建括号和光标位置
- **经典计算器 1/x 和 |x|**：原实现覆盖整个表达式。修复：改用 `#0` 占位符模式 `\frac{1}{#0}` 和 `\left|#0\right|`
- **科学计算器无值希腊字母替换**：α/β/γ/δ/θ/λ 6 个无实际值的希腊字母按钮替换为 12 个有实际值的科学常数
- **新增科学常数按钮**：π、e、c（光速）、h（普朗克）、ħ（约化普朗克）、R（气体常数）、N_A（阿伏伽德罗）、k_B（玻尔兹曼）、m_e（电子质量）、e（元电荷）、ε₀（真空介电常数）、σ（Stefan-Boltzmann）
- **models.js 新增常量**：`me = 9.1093837015e-31`（电子质量）、`sigmaSB = 5.670374419e-8`（Stefan-Boltzmann）
- **latexToMathJS 增强**：新增 `\hbar`、`\varepsilon_{0}`、`\sigma_{SB}`、`e_{Charge}`、`k_{B}`、`N_{A}`、`m_{e}` 及 MathLive `\mathrm{}` 格式的转换
- **normalizeExpression 修复**：`\\ln(` → `log(` （匹配 math.js 自然对数），添加 `#0` 占位符清理
- **修改文件**：`index.html`（按钮重写）、`app.js`（转换函数增强）、`models.js`（常量 + 解析修复）

## 符号微积分模块（nerdamer 集成）
- **新增库**：nerdamer v1.1.13，通过 jsDelivr CDN 动态加载（core → Algebra → Calculus → Solve）
- **新增标签**：科学计算器 "Symbolic Calculus" 标签页，4 个子模块
- **子模块功能**：
  - **Symbolic Derivative**：符号求导 dⁿf/dxⁿ，支持任意阶
  - **Partial Derivative**：偏导数 ∂ⁿf/∂xⁿ，支持 "All First Partials" 自动检测变量
  - **Symbolic Integration**：不定积分 ∫f(x)dx + C 和定积分 ∫[a,b]f(x)dx
  - **Differential Equation**：一阶 ODE dy/dx = f(x,y)，支持分离变量法自动求解
- **结果渲染**：使用 MathJax 渲染 LaTeX 公式
- **修改文件**：`index.html`（面板 HTML）、`app.js`（事件逻辑 + latexToNerdamer 转换函数）、`main.js`（CDN 加载）
