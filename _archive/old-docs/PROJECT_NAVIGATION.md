## 💡 高效工作技巧

### 技巧1：分屏工作
```
推荐使用三栏布局：

┌──────────────┬──────────────┬──────────────┐
│              │              │              │
│  浏览器      │  TESTING_    │  BUG_        │
│  (测试应用)  │  PLAN.md     │  TRACKER.md  │
│              │  (测试清单)  │  (问题记录)  │
│              │              │              │
└──────────────┴──────────────┴──────────────┘

工具推荐：
- Windows: PowerToys FancyZones
- Mac: Rectangle, Magnet
- Linux: i3, Awesome WM
```

### 技巧2：快捷键速查
```
浏览器DevTools:
  F12         - 打开开发者工具
  Ctrl+Shift+C - 元素选择器
  Ctrl+Shift+J - 直接打开Console
  Ctrl+Shift+M - 设备模拟模式

编辑器:
  Ctrl+F      - 搜索
  Ctrl+H      - 替换
  Ctrl+/      - 注释
  Ctrl+S      - 保存

系统:
  Win+Shift+S - 截图 (Windows)
  Cmd+Shift+4 - 截图 (Mac)
  Alt+Tab     - 切换窗口
```

### 技巧3：批量处理
```
相似测试一起做：
✓ 所有一次运算 (加减乘除)
✓ 所有三角函数 (sin, cos, tan)
✓ 所有对数函数 (ln, log, log10)
✓ 所有单位切换 (MPa↔GPa, μm↔mm↔m↔nm)

提高效率，减少上下文切换
```

### 技巧4：定期保存进度
```
每完成一个模块：
1. 更新 TESTING_PLAN.md 的勾选状态
2. 在 BUG_TRACKER.md 记录新问题
3. Git提交（如果使用版本控制）
4. 休息5分钟 ☕

避免疲劳导致的遗漏
```
