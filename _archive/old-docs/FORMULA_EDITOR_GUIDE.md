# 科学计算器增强功能使用指南

## 🎯 新增功能概览

本次更新为科学计算器添加了以下核心功能：

### 1. MathType 式公式编辑器 ✅
- **专业的数学公式输入体验**
- 支持 LaTeX 格式的可视化编辑
- 实时渲染，所见即所得
- 鼠标单击即可直接输入公式

### 2. 公式复制功能 📋
支持三种格式导出：
- **LaTeX 格式** - 适用于学术论文、LaTeX 文档
- **MathML 格式** - 适用于网页、XML 应用
- **纯文本格式** - 适用于简单文本编辑器

### 3. OCR 图片识别公式 📷
- 从图片中自动识别数学公式
- 支持手写和印刷体公式识别
- 一键转换为可编辑的 LaTeX 格式

### 4. 增强的虚拟键盘 ⌨️
包含以下分类的符号按钮：
- **基础数学结构**：分数、根号、幂次、下标
- **高级数学结构**：求和 Σ、积分 ∫、极限 lim、偏导数 ∂
- **三角函数**：sin, cos, tan, arcsin, arccos, arctan
- **对数和指数**：ln, log, exp, |x|, 组合数 C(n,k), 阶乘 n!
- **希腊字母**：α, β, γ, δ, θ, λ, μ, σ, ω, Δ, Σ, Ω
- **数学符号**：±, ×, ÷, ∞, ·
- **材料科学符号**：ε (应变), ρ (密度), η (粘度), φ (体积分数), ψ (波函数)
- **常数**：π, e, i (虚数单位)

## 📖 使用教程

### 一、基本公式输入

#### 方法 1：使用虚拟键盘
1. 点击对应的符号按钮（如 `sin`, `√`, `a/b`）
2. 在占位符位置输入参数
3. 按 Tab 键或方向键移动到下一个占位符

**示例：输入二次方程求根公式**
```
步骤：
1. 点击 "a/b" 按钮 → 显示分数模板
2. 在分子位置输入：-b ± √(b² - 4ac)
   - 先输入 "-b"
   - 点击 "±" 按钮
   - 点击 "√x" 按钮
   - 输入 "b^2 - 4*a*c"
3. 按 Tab 键切换到分母
4. 输入 "2a"
```

#### 方法 2：直接键盘输入
1. 点击表达式输入框获得焦点
2. 直接输入 LaTeX 代码
3. MathLive 会实时渲染公式

**常用 LaTeX 命令：**
```latex
\frac{a}{b}      # 分数 a/b
\sqrt{x}         # 平方根
x^{n}            # x 的 n 次方
x_{i}            # x 下标 i
\sin(x)          # 正弦函数
\alpha           # 希腊字母 α
\pi              # 圆周率 π
\sum_{i=1}^{n}   # 求和符号
\int_{a}^{b}     # 积分符号
```

#### 方法 3：混合输入
- 可以先用虚拟键盘插入结构
- 然后用键盘快速输入变量名和数字
- 结合两种方式提高效率

---

### 二、复制公式

#### 步骤：
1. 在表达式输入框中输入或编辑公式
2. 点击下方的工具栏按钮：
   - **📋 LaTeX** - 复制 LaTeX 格式
   - **📄 MathML** - 复制 MathML 格式
   - **📝 Text** - 复制纯文本格式
3. 粘贴到目标应用程序

#### 应用场景：
- **LaTeX**：Word (Equation Editor)、LaTeX 文档、Overleaf
- **MathML**：网页开发、XML 应用、某些学术软件
- **Text**：简单文本编辑器、聊天软件、邮件

---

### 三、OCR 图片识别公式

#### 步骤：
1. 准备包含数学公式的图片（PNG、JPG 格式）
2. 点击 **📷 OCR** 按钮
3. 选择图片文件
4. 等待识别完成（约 1-3 秒）
5. 识别结果会自动填入表达式输入框

#### 注意事项：
- **演示模式**：当前版本未配置真实 OCR API，会插入示例公式
- **生产环境**：需要注册 Mathpix API 或其他 OCR 服务
- **图片质量**：建议使用清晰的印刷体公式图片
- **手写识别**：部分 OCR 服务支持手写公式识别

#### 如何配置真实 OCR API：
1. 注册 [Mathpix](https://mathpix.com/) 账号
2. 获取 `app_id` 和 `app_key`
3. 在 `app.js` 中找到 `recognizeFormula()` 函数
4. 将 `hasApiKey` 设置为 `true`
5. 替换 `YOUR_APP_ID` 和 `YOUR_APP_KEY` 为实际值

```javascript
// 修改前
const hasApiKey = false;

// 修改后
const hasApiKey = true;

// 替换 API 密钥
'app_id': 'your_actual_app_id',
'app_key': 'your_actual_app_key'
```

### 四、虚拟键盘使用技巧

#### 1. 颜色编码
不同颜色的按钮代表不同类型：
- **蓝色**：基础运算符 (+, −, ×, ÷)
- **绿色**：函数 (sin, cos, ln, log)
- **紫色**：常数 (π, e, i)
- **青色**：希腊字母 (α, β, γ, ...)
- **橙色**：数学符号 (±, ×, ÷)
- **粉色**：材料科学符号 (ε, ρ, η, φ, ψ)
- **深蓝色**：数学结构 (分数、根号、求和等)

#### 2. 占位符导航
- 带有 `#?` 标记的按钮会创建占位符
- 使用 **Tab 键** 或 **方向键** 在占位符间移动
- 光标会自动定位到第一个空位

#### 3. 快捷操作
- **Clear**：清除全部输入
- **⌫**：删除前一个字符
- **=**：计算结果

## 🔧 常见问题

### Q1: 鼠标单击后无法输入公式？
**A:** 请确保：
1. 浏览器已完全加载页面（检查控制台是否有错误）
2. MathLive 库已正确加载（查看网络请求）
3. 尝试强制刷新浏览器（Ctrl + Shift + R）
4. 检查是否使用了支持的浏览器（Chrome、Firefox、Edge 最新版）

### Q2: 公式显示不正确？
**A:** 
- 确保使用的是 LaTeX 格式
- 检查括号是否匹配
- 避免使用中文标点符号
- 参考上面的 LaTeX 命令列表

### Q3: OCR 识别失败？
**A:**
- 当前版本是演示模式，需要配置真实 API
- 图片质量可能不够清晰
- 尝试使用印刷体公式而非手写体
- 检查网络连接是否正常

### Q4: 如何保存常用公式？
**A:**
- 可以使用浏览器的书签功能保存整个页面状态
- 或者将 LaTeX 代码复制到文本文件中保存
- 未来版本会添加公式收藏功能

### Q5: 虚拟键盘不显示？
**A:**
- 检查 CSS 是否正确加载
- 确认没有 JavaScript 错误
- 尝试调整浏览器缩放比例（100%）
- 清除浏览器缓存后重试

## 📚 进阶使用

### 材料科学公式示例

#### 1. Hall-Petch 关系（晶界强化）
```latex
\sigma_y = \sigma_0 + \frac{k_y}{\sqrt{d}}
```
输入方法：
1. 输入 `\sigma_y = \sigma_0 +`
2. 点击 "a/b" 按钮
3. 分子输入 `k_y`
4. 分母输入 `\sqrt{d}`

#### 2. Orowan 强化（析出强化）
```latex
\Delta\tau = \frac{Gb}{L}
```
输入方法：
1. 输入 `\Delta\tau =`
2. 点击 "a/b" 按钮
3. 分子输入 `Gb`
4. 分母输入 `L`

#### 3. 固溶强化
```latex
\Delta\sigma = k c^{n}
```
输入方法：
1. 输入 `\Delta\sigma = k`
2. 点击 "xⁿ" 按钮
3. 底数输入 `c`
4. 指数输入 `n`
