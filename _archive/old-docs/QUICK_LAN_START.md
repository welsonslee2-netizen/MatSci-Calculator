# 🚀 局域网访问 - 3 步快速开始

## ⚡ 超简单！只需 3 步

### 第 1 步：配置防火墙（仅首次）

**右键点击** `setup-firewall.bat` → **以管理员身份运行**

等待看到 "✅ 防火墙规则添加成功！"

---

### 第 2 步：启动服务器

**双击运行** `start-lan-server.bat`

会看到类似这样的输出：
```
✅ 本机 IP: 100.64.123.45

📍 本地访问: http://localhost:8080
📱 手机访问: http://100.64.123.45:8080
💻 其他电脑: http://100.64.123.45:8080
```

**记住这个 IP 地址！**（例如：100.64.123.45）

---

### 第 3 步：在手机/其他电脑上访问

1. 确保设备已连接 **Tailscale**
2. 打开浏览器
3. 输入：`http://你的IP:8080`
   - 例如：`http://100.64.123.45:8080`
4. 按 Enter

**完成！** 🎉

---

## 📱 测试重点

在手机上特别测试：

- ✅ 小键盘按钮点击（sin, cos, √x, 数字等）
- ✅ 输入框是否能正常输入
- ✅ 页面布局是否正常

---

## ❓ 无法访问？

1. **检查 Tailscale**：确保两台设备都已连接
2. **检查防火墙**：重新运行 `setup-firewall.bat`
3. **检查 IP**：确认输入的 IP 地址正确
4. **检查服务器**：在本机访问 `http://localhost:8080` 测试

详细故障排查请查看：[LAN_ACCESS_GUIDE.md](LAN_ACCESS_GUIDE.md)

---

## 🛑 停止服务器

在命令行窗口按 **Ctrl+C**
