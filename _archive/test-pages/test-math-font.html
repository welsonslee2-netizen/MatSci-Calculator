⚡ 实时格式化演示

支持的转换：
• ^2, ^3, ^n → 上标格式（², ³, ⁿ）
• pi → π
• sqrt( → √(
• times → ×
• div → ÷
• leq → ≤, geq → ≥
• neq → ≠, approx → ≈
• infinity → ∞

🎯 技术实现

通过以下方式实现：

  1. CSS 变量定义数学字体栈：--font-math
  2. JavaScript 实时监听输入事件
  3. 正则表达式匹配并替换为上标/特殊符号
  4. 保持光标位置不变，避免输入中断
